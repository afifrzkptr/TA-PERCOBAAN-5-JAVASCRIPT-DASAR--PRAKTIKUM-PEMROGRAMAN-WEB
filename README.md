# TA-PERCOBAAN-5-JAVASCRIPT-DASAR--PRAKTIKUM-PEMROGRAMAN-WEB
# Kalkulator

Repositori ini berisi implementasi Tugas Akhir (TA) untuk Praktikum Pemrograman Web (Percobaan 5: JavaScript Dasar), yaitu membuat kalkulator interaktif dengan fitur-fitur lanjutan.

## Fitur Utama

Aplikasi kalkulator ini dikembangkan menggunakan HTML, CSS (Tailwind CSS dan Neumorphism), dan JavaScript murni, dan mencakup fitur-fitur berikut:

* **Antarmuka Neumorphism:** Desain modern dengan efek bayangan lembut (soft UI).
* **Perhitungan Berantai:** Mendukung urutan operasi matematika yang kompleks (misal: 10 + 5 * 2).
* **Penanganan Error:** Mendeteksi dan menampilkan pesan *error* yang jelas untuk Pembagian dengan Nol (`Error: Dibagi 0`).
* **Dukungan Keyboard:** Dapat dioperasikan menggunakan tombol angka, operator, Enter, dan Backspace pada keyboard.
* **Riwayat (History):** Menyimpan 5 perhitungan terakhir.
* **Fungsi Memori (M+, M-, MR, MC):** Untuk menyimpan dan memanggil nilai secara terpisah.

## Teknologi

* **HTML5:** Struktur dasar.
* **CSS3:** Styling (menggunakan variabel CSS untuk Neumorphism).
* **JavaScript (ES6+):** Logika kalkulator dan manipulasi DOM.
* **Tailwind CSS:** Untuk *utility classes* dan *responsive layout*.
